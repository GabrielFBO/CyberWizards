🧙‍♂️ CyberWizards

Landing Page moderna e interativa para uma "academia virtual" de desenvolvimento web, com foco em conversão de leads e experiência visual imersiva (tema cyber + magia).

---

🚀 Acesse o projeto

🔗 https://cyber-wizards.vercel.app/

---

🧠 Sobre o projeto

O CyberWizards foi desenvolvido como um projeto prático com o objetivo de simular uma landing page real utilizada por empresas para captação de leads.

O foco não foi apenas no design, mas também em:

- Experiência do usuário (UX)
- Estrutura de conversão
- Rastreamento de dados (Analytics)
- Deploy em ambiente real

---

⚙️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Analytics 4 (GA4)
- Vercel (Deploy)

---

🎯 Funcionalidades

- Landing page responsiva  
- Identidade visual personalizada (cyber + magia)  
- Formulário com validação  
- Redirecionamento para página de confirmação  
- Página de "Obrigado"  
- Botão flutuante do WhatsApp  
- Rastreamento de eventos (GA4)  
- Open Graph (preview para compartilhamento)  
- Favicon personalizado  

---

📊 Tracking e Analytics

O projeto utiliza Google Analytics 4 para rastrear:

- Visualizações de página
- Interações com formulário
- Conversão de leads

---

🧪 Aprendizados

Durante o desenvolvimento, foram trabalhados conceitos importantes como:

- Estrutura de landing pages
- Otimização de conversão (CTA, layout, copy)
- Implementação de tracking (GA4)
- Debug de eventos duplicados
- Deploy e ambiente de produção
- Boas práticas de front-end
